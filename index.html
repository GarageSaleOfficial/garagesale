What is G-Sale?

G-Sale is an app that utilizes AI techology to help you decide what and when to sell by comparing different prices using our cloud system. This ensures you the best value for your merchandise!

Why G-Sale?

  • Advance Machine Learning

    We used advance machine learning techniques to help boost the speed and accuracy of our app.

  • Simple Interface

    Our minimalistic user interface is designed to be simple. A few simple buttons gives you complete access to our variety of tools.

  • Fully Customizable

    We included all the settings the user needs to fully customize their experience to their liking.

G-Sale by the numbers

G-Sale is an app created by a team of 4 consisting of Jack Luo, Kevin Chu, Alec Chen and Wesley Hung. We created this application at Code Day, Bay Area in under 24 hours. We used Android Studio, Java, and XML to create the app. HTML, CSS and JS were used for the production of the website itself.
